# 2026-07-01

## UI Redesign — Apple Design System (DESIGN-apple.md)

遵循 DESIGN-apple.md 对 Markor 应用进行 UI 翻新，采用 Apple 设计语言：

### 修改的文件

**色彩体系（Foundation）：**
- `res/values/colors.xml` — 全新色板：Action Blue #0066CC 单一强调色、Ink #1D1D1F 主文字、Parchment #F5F5F7 背景
- `res/values-night/colors.xml` — 暗色模式适配，Sky Link Blue #2997FF 用于深色文件夹图标
- `res/values/styles.xml` — 更新主题属性，状态栏纯黑，新增卡片/按钮样式
- `res/values/dimens.xml` — Apple 间距体系 (8/12/17/24dp)，18dp 圆角

**Drawable 资源：**
- `res/drawable/card_item_background.xml` — 新增：白色卡片 + 18dp 圆角 + hairline 边框
- `res/drawable/rounded_input_background.xml` — 新增：11dp 圆角输入框背景
- `res/drawable/rounded_corner_background.xml` — 更新：白色填充 + hairline 边框
- `res/drawable/bottomnav_color_selector.xml` — 更新：选中态 Action Blue，未选中态 secondary_text

**布局文件：**
- `res/layout/opoc_filesystem_item.xml` — 卡片化列表项：白色圆角卡片 + 左侧蓝色引导条 + 优化间距
- `res/layout/main__activity.xml` — 底部导航改为白色背景 + 顶部 hairline 分割线
- `res/layout/new_file_dialog.xml` — 圆角输入框 + 优化标签颜色
- `res/layout/rename__dialog.xml` — 圆角输入框 + 优化间距
- `res/layout/document__fragment__edit.xml` — 编辑器改为白色 (canvas) 背景，移除阴影
- `res/layout/opoc_filesystem_dialog.xml` — 更新按钮文字颜色至 accent
- `res/layout/file_info_dialog.xml` — 更新标签文字颜色

### 核心设计原则
- 单一强调色 #0066CC
- 无装饰性渐变/阴影
- 18dp 圆角卡片 + hairline 边框
- Parchment 暖灰底 + Ink 近黑文字

## 自定义 MD 预览主题

增加了 Markdown WebView 预览主题选择功能，用户可在「Settings → View Mode → Markdown Preview Theme」中切换。

### 新增主题

| 主题 | 亮色背景 | 暗色背景 | 特点 |
|------|---------|---------|------|
| **Auto** (默认) | 白底 #303030 文字 | #303030 底白字 | 跟随系统暗色模式 |
| **Sepia** | #FBF0D9 暖纸色 | #2B2416 深棕 | 暖色调护眼 |
| **Solarized** | #FDF6E3 | #002B36 | 经典低对比度配色 |
| **Nord** | #ECEFF4 | #2E3440 | 北欧冷色系 |
| **Gruvbox** | #FBF1C7 | #282828 | 复古大地色 |
| **Dracula** | — | #282A36 固定暗色 | 编程主题风格 |
| **AMOLED** | — | #000000 纯黑 | OLED 省电 |

### 修改的文件
- `res/values/string-not_translatable.xml` — 新增 pref_key__markdown_view_theme
- `res/values/strings.xml` — 主题显示名称
- `res/values/arrays.xml` — 主题选择列表
- `res/xml/preferences_master.xml` — View Mode 设置项
- `java/.../model/AppSettings.java` — getViewModePreviewTheme()
- `java/.../format/TextConverterBase.java` — 新增 getPreviewThemeCss() + isPreviewThemeEffectivelyDark()，修改 putContentIntoTemplate()
- `java/.../format/markdown/MarkdownTextConverter.java` — Prism.js 和 Mermaid 主题跟随预览主题选择
